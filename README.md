# AutoEarworm

This is a tool to generate an "On Repeat" like playlist based on your top songs from [last.fm](https://last.fm).

You need [node.js](https://nodejs.org/en) to run this; fill in the appropriate details in `getSongs.js` and `getToken.js` (as well as the optional stuff in `playlist.js` if you wish) and then type `node .` into the console and follow the instructions!
